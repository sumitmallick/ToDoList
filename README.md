# Pyrello

Simple Trello clone made in Python

Requirements:
- [Docker][docker]
- [Python][python] (>= 3.7)
- [Git][git]
- [PostgreSQL][postgresql]

## Usage

Run app
```
./up.sh
```

Stop app
```
./down.sh
```

Print logs
```
./logs.sh
```
